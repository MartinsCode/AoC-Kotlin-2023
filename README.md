## Getting Started

- Copy Day25.kt to DayXX.kt, switch the names of the input files.
- Create DayXX.txt and DayXX_test.txt and put in the testdata and the "real" 
data.
- Rename file in DayXX.kt to be read in.
- Run file. 
- Read solution.
